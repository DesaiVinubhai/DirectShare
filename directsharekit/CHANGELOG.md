## [1.0.0] - 2026-03-20

### Added
- Initial public DirectShareKit package
- `ShareManager` with single/multiple sharing methods for:
  - WhatsApp
  - Telegram
  - Messenger
  - Gmail
  - Google Drive
  - Generic Android chooser
- App detection helpers:
  - `isWhatsAppInstalled`
  - `isTelegramInstalled`
  - `isMessengerInstalled`
  - `isGmailInstalled`
  - `isDriveInstalled`
- `FileRepository`, `ShareUseCase`, utility helpers
- Built-in `FileProvider` + secure path XML
- Consumer ProGuard rules
